Using XIP

Installing XploR using XIP

We recommend using xip to download and install XploR as it is the most convenient maintain and upgrade the IDE, as well as to access additional related components.

Download and install XploR Studio IDE by the following command:

For Linux

  xip install xplor-studio-ide-linux64
For Windows

  xip install xplor-studio-ide-win64
